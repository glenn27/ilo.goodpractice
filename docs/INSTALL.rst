ilo.goodpractice Installation
-----------------------------

To install ilo.goodpractice using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``ilo.goodpractice`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        ilo.goodpractice

* Re-run buildout, e.g. with:

    $ ./bin/buildout
